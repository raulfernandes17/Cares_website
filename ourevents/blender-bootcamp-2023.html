Frequently asked questions (FAQ)

  • To download Blender Visit the official Blender website : www.blender.org
  • Navigate to the “Download” section.
  • Choose your operating system (Windows, macOS, Linux) and download the appropriate version.
  • Install Blender by running the downloaded installer.

  • Blender is a powerful 3D creation suite used for modeling, animation, sculpting, texturing, rendering, and more.

  • Middle mouse button: Rotate the view.
  • Scroll wheel: Zoom in and out..
  • Shift + Middle mouse button: Pan the view.

  • The interface includes a 3D viewport, toolbars, panels, and menus.
  • The Properties panel on the right adjusts object properties
  • The Outliner lists all objects in the scene.
  • The Timeline manages animations and keyframes.

  • Press Shift + A in the viewport to access the Add menu. Choose the desired primitive shape (cube, sphere, cylinder, etc.).

  • Select both objects by holding Shift and clicking them.
  • Press Ctrl + J to join them into a single object.

  • Select an object, press G to grab, then type values or use the mouse.
  • Alternatively, use the Transform panel in the N-key menu to adjust position numerically.

  • Lighting affects how objects are illuminated in a scene.
  • Add lights like Point, Sun, or Area lights from the Add menu.
  • Adjust light strength, color, and position in the Properties panel.

  • Rendering is the process of generating a final image or animation from a 3D scene.
  • Adjust render settings in the Properties panel.
  • Click “Render” to create the final output.

  • Click on file on the top menu bar and then click on Save As
  • Select the appropriate location, put the name of the file eg. myModel.blend and then click on the ‘ Save As ’ button.

  • Click on render on the top menu bar
  • Click on render image, a new window will appear with your 3D model
  • Click on Image on the menu bar and then click on Save As
  • Select the appropriate location, put the name of the file eg. myModel.png and then click on the ‘ Save As Image ‘ button.
